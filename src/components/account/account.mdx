# Account

O componente `Account` é responsável por gerenciar o estado do usuário na aplicação, realizando a busca dos dados do usuário de forma assíncrona e armazenando no estado global.

## Funcionamento

Este componente utiliza o Redux Toolkit para gerenciar o estado do usuário e a RTK Query para carregar os dados do usuário. Ele funciona de forma passiva, ou seja, não renderiza nada na tela, mas mantém o estado atualizado em segundo plano.

### Comportamento
- **Verifica se há um usuário no estado global**.
- **Se não houver**, dispara uma requisição para buscar os dados do usuário.
- **Quando os dados são carregados com sucesso**, armazena o usuário no Redux.

## Exemplo de Uso

O componente `Account` deve ser incluído em uma das camadas superiores da aplicação, como dentro do `layout.tsx` ou `_app.tsx` (dependendo da estrutura do projeto), para garantir que os dados do usuário sejam carregados e gerenciados corretamente:

```tsx
import Account from '@components/account'

const Layout = () => (
  ...
  <Account />
  ...
)

export default Layout
```

## Tecnologias Utilizadas
- **Redux Toolkit** para gerenciar o estado global.
- **RTK Query** para fazer a requisição de dados do usuário.
- **React Hooks (`useEffect`)** para disparar a busca e atualizar o estado automaticamente.
